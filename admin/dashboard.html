Configuración de Correo (SMTP)

Configura el servidor de correo para recibir notificaciones cuando un cliente envíe un mensaje de contacto.

Cómo configurar:
• Gmail: Host: smtp.gmail.com, Puerto: 587. Necesitas una "App Password" (no tu contraseña normal).
• Outlook: Host: smtp-mail.outlook.com, Puerto: 587.
• Yahoo: Host: smtp.mail.yahoo.com, Puerto: 587.
• Otro: Consulta con tu proveedor de correo los datos SMTP.
